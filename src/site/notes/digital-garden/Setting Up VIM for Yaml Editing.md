# Setting Up VIM for Yaml Editing

```plain
set expandtab
set tabstop=2
set shiftwidth=2
```

```
:set expandtab shiftwidth=2 tabstop=2" 
# Or the short form 
:set et sw=2 ts=2

:set cursorcolumn
:set cursorline
```
